## 🇬🇧 English Version

### 📌 About the Project
This project **will resurrect** the core gameplay of the legendary **Zombs.io**. The game **will feature** a charming, nostalgic **pixel-art aesthetic**, completely replacing the original flat and smooth graphics.

### ⚙️ Current Production Progress
- [x] Basic game engine & pixel-art graphics concept
- [ ] Classic Survival mechanics (base building, gold mines, zombie waves)
- [ ] LAN Local Co-op mode
- [ ] New mechanics (blocks, upgrade tiers, weapons, enemies)
- [ ] Pets system
- [ ] Expanded Game Modes (PvP, destroying bases, up to 5-player teams)

### 🎮 Initial Release Features
* **Classic Survival:** You **will be able to** build your ultimate base, defend your gold mines, and survive increasingly difficult nightly zombie waves.
* **LAN Local Co-op:** You **will be able to** team up with your friends locally over LAN to survive together. *(Note: Online remote multiplayer will not be supported initially, but is planned for the future).*

### 🚀 Future Roadmap
In its early versions, the game **will be** a typical survival experience. However, we plan to expand the project significantly:
* **New Mechanics:** Brand new blocks, higher upgrade tiers, advanced weapons, and unique enemies.
* **Pets System:** Loyal companions that **will help** you defend your base and gather resources.
* **Expanded Game Modes:** We **will reintroduce** the original full PvP mechanics where every player can be an enemy, raid others' bases, or form tactical teams of up to 5 players.

## 🇵🇱 Wersja Polska

### 📌 O projekcie
Gra **będzie miała na celu** przywrócenie do życia kultowego **Zombs.io**. Zyska ona zupełnie nowy charakter dzięki klimatycznej **grafice pixel art**, która zastąpi oryginalny, gładki i prosty styl wizualny.

### ⚙️ Aktualny postęp produkcji
- [x] Koncept graficzny pixel art i przygotowanie silnika gry
- [ ] Mechaniki klasycznego survivalu (budowa bazy, kopalnie złota, fale zombie)
- [ ] Tryb lokalnej kooperacji (LAN)
- [ ] Nowe mechaniki (bloki, tiery ulepszeń, bronie, wrogowie)
- [ ] System Petów
- [ ] Rozbudowane tryby gry (PvP z niszczeniem baz, 5-osobowe drużyny)

### 🎮 Funkcje pierwszych wersji
* **Klasyczny Survival:** **Będziesz mógł** budować bazę, bronić kopalni złota i przetrwać coraz trudniejsze, nocne fale zombie.
* **Lokalna kooperacja (LAN):** **Będziesz mógł** grać ze znajomymi w sieci lokalnej LAN, aby wspólnie walczyć o przetrwanie. *(Uwaga: Zdalny sieciowy multiplayer nie będzie dostępny na samym początku, ale jest w planach na przyszłość).*

### 🚀 Plany na przyszłość
W pierwszych wersjach gra **będzie** typowym survivalem, jednak w przyszłości projekt **zostanie rozwinięty** o:
* **Nową zawartość:** Całkowicie nowe bloki, wyższe tiery ulepszeń, potężniejsze bronie oraz unikalnych przeciwników.
* **System Petów:** Wierni towarzysze, którzy **pomogą** Ci w obronie bazy i zbieraniu surowców.
* **Rozbudowane tryby gry:** Powrót do oryginalnego PvP, w którym każdy gracz może być wrogiem. **Pojawi się** opcja niszczenia baz innych graczy oraz tworzenia drużyn (maksymalnie do 5 osób).
